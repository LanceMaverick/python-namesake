## namesake
### A dead simple, memorable name generator for Python

#### Installation
```sh
pip install namesake
```

#### Usage
```python
#!/usr/bin/env python
from namesake import getname
print(getname())
```

A command line version is also included . . .
```bash
#!/usr/bin/env bash
namesake-name
```

Python 2.6, 2.7, 3.3, 3.4, 3.5 supported
